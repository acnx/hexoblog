C语言学习笔记

发表于 2020-05-22 | 本文阅读量次 | 分类于 编程语言基础 ， C语言

C语言是一门面向过程的、抽象化的通用程序设计语言，广泛应用于底层开发。C语言能以简易的方式编译、处理低级存储器。C语言是仅产生少量的机器语言以及不需要任何运行环境支持便能运行的高效率程序设计语言。尽管C语言提供了许多低级处理的功能，但仍然保持着跨平台的特性，以一个标准规格写出的C语言程序可在包括类似嵌入式处理器以及超级计算机等作业平台的许多计算机平台上进行编译。

第一章 初识C语言

1.1 C语言的起源

1972年，贝尔实验室的丹尼斯·里奇（Dannis Ritchie）和肯·汤姆逊（Ken Thompson）在B语言的基础上进行设计

1.2 选择c语言的理由

  1. 设计特性
  2. 高效性
  3. 可移植性
  4. 强大而灵活
  5. 面向程序员
  6. 缺点：指针

1.3 C语言的应用范围

  1. 系统
  2. 程序
  3. 编译器
  4. 游戏

1.4 计算机能做什么

从内存中获取并执行一条指令，然后再从内存中获取并执行下一条指令

两个有趣的知识：

  1. 存储在计算机的所有内容都是数字；
  2. 计算机程序最终必须以机器语言来表示。

1.5 高级计算机语言和编译器

高级语言以更抽象的方式描述行为，不受限于CPU或指令集，但需要编译器翻译成机器语言。

1.6 语言标准

  1. 1978年，《The C Programming Language》 K&R Brian Kernighan和Dannis Ritchie 称为K&R C或经典C
  2. 1983-1989年，美国国家标准协会 ANSI C
  3. 1990年，国际化标准组织采用。ISO C 和ANSI C完全相同，称ANSI/ISO C 通常叫做C89 （ANSI1989年批准）或C90 （ISO1990年批准）
  4. 1994年 ，ANSI/ISO联合会修订，发布C99标准
  5. 2011年，标准委员会发布 C11标准

1.7 使用C语言的7个步骤

  1. 定义程序的目标
  2. 设计程序
  3. 编写代码
  4. 编译
  5. 运行程序
  6. 测试和调试程序
  7. 维护和修改程序
