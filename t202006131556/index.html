曲线救国：间接实现301跳转

发表于 2020-06-13 | 本文阅读量次

起因

今天想把手里的一个万网域名B解析到博客A，做一个301跳转，却看到有限制，必须是是备案且接入商是万网的域名A才可以跳转。

思路

把域名B解析到 github pages，再从 github pages做跳转到A。

实施

  1. 在阿里云控制面板将域名B解析到github。

  1. 在 github填写自己的域名B

  2. 在 index.html中做跳转，至域名A

完成

现在，你可以使用域名B访问域名A了。

总结

只是小小的玩耍
